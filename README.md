# README

Rails 7 with Bootstrap 5 and Devise.


### Template setup commands: 

* `rails new ...`
* add gems: gem 'devise', gem 'bootstrap', gem 'sassc-rails'
* `rails g devise:install`
* `rails g devise:user`
* `bin/importmap pin bootstrap`
* `bundle i`


### To startup:

* `rails db:migrate`
* `rails s`
